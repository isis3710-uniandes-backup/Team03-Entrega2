# Team03-Entrega2

## Para desplegar la aplicación:
- Moverse a la carpeta raiz app-back
- Ejecutar npm install en consola
- Moverse a la carpeta front
- Ejecutar npm install en consola
- Volver a la carpeta raiz app-back
- Ejecutar npm start
